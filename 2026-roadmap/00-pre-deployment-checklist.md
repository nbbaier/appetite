# Pre-Deployment Checklist

Remaining items needed before a production launch. Salvaged from the now-removed
Task Master "Pre-Deployment Checklist" task (everything else in that backlog was
already complete and reflected in the codebase).

## Open items

- [ ] **Screenshots** — Capture high-quality screenshots of key features for docs,
  marketing, and the app store: dashboard, recipe search, pantry management,
  shopping lists, and the AI chat assistant.
- [ ] **Visual polish** — Final UI refinements: animations, spacing, colors,
  typography, and overall visual consistency throughout the app.
- [ ] **Badges** — Add notification badges, status indicators, and other visual
  badges across the interface.
- [ ] **One-liner elevator pitch** — Craft a concise, memorable sentence capturing
  the app's core value proposition.
- [ ] **App description** — Write a detailed description (features, benefits, use
  cases) for marketing and documentation.
- [ ] **Demo video** — Produce a high-quality video demonstrating key features and
  user workflows.
- [ ] **Deploy** — Execute the final production deployment with proper
  configuration and monitoring.
- [ ] **README** — Write comprehensive README documentation: setup instructions,
  features overview, and usage examples.
- [ ] **Remove avatar upload** — Clean up the avatar upload UI components and
  related functionality that are not needed.
